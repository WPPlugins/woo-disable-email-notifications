== Description ==

With this email you will be able to choose which email notifications you want to disable on Woocommerce.

== Installation ==

1. Upload the folder `wocommerce-show-stock` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it!
